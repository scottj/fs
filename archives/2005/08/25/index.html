Movable Type 3.2 is Finally Here

Almost two months after I prematurely declared that Movable Type 3.2 was released, it is now here. Six Apart has the details. I actually saw Anil’s announcement first. But it was Brad’s post that really sparked my attention today:

The just-released Movable Type 3.2 includes both an OpenID server and consumer plugin. They’re not enabled by default, but this is a good first step.
This seems like a step in the right direction. I know what I will be doing tonight.
